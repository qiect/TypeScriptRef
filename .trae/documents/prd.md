## 1. 产品概述

TypeScript 教程与速查网站是一个面向前端开发者的高质量学习平台，提供从入门到高级的 TypeScript 系统教程和快速参考速查表。帮助开发者快速掌握 TypeScript 核心概念，并在日常开发中通过速查表快速查找语法和用法。

- 目标用户：有 JavaScript 基础、希望学习或深入 TypeScript 的前端开发者
- 核心价值：系统化学习路径 + 即查即用的速查卡片，一站式解决 TypeScript 学习与日常查阅需求

## 2. 核心功能

### 2.1 功能模块

1. **首页**：Hero 区域、特性展示、教程导航入口、速查表入口
2. **教程页面**：分章节的 TypeScript 系统教程，包含代码示例与语法高亮
3. **速查表页面**：按主题分类的速查卡片，支持搜索与筛选
4. **搜索功能**：全局搜索教程内容和速查条目

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | Hero 区域 | 大标题、副标题、CTA 按钮，展示网站核心定位 |
| 首页 | 特性展示 | 3-4 个特性卡片，展示网站核心卖点 |
| 首页 | 教程导航 | 教程章节列表，点击进入对应教程 |
| 首页 | 速查入口 | 速查表分类入口，快速跳转 |
| 教程页面 | 章节侧边栏 | 左侧目录导航，支持章节切换 |
| 教程页面 | 内容区域 | 教程正文、代码示例、语法高亮 |
| 教程页面 | 上下章节导航 | 底部上一章/下一章导航 |
| 速查表页面 | 分类筛选 | 按主题分类的标签筛选 |
| 速查表页面 | 搜索栏 | 实时搜索速查条目 |
| 速查表页面 | 速查卡片 | 代码片段 + 说明的卡片式展示 |

## 3. 核心流程

用户进入首页 → 浏览教程目录或速查表入口 → 点击进入教程章节阅读学习内容 → 或进入速查表搜索/筛选所需语法 → 查看代码示例和说明

```mermaid

    "选择章节" --> "阅读教程内容"
    "阅读教程内容" --> "切换章节"
    "浏览速查表入口" --> "搜索/筛选"
    "搜索/筛选" --> "查看速查卡片"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝 (#1a1b2e) + 亮蓝 (#3b82f6) 强调色，搭配 TypeScript 蓝 (#3178c6)
- 辅助色：暗灰背景 + 白色文字，代码块深色主题
- 按钮风格：圆角按钮，hover 时有微妙的发光效果
- 字体：代码使用 JetBrains Mono，正文使用 Noto Sans SC
- 布局风格：左侧固定导航 + 右侧内容区域，卡片式速查展示
- 图标风格：使用 Lucide 图标库，线条风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | Hero 区域 | 渐变背景、大标题动画、TypeScript Logo、CTA 按钮 |
| 首页 | 特性展示 | 3 列卡片、图标 + 标题 + 描述、hover 上浮效果 |
| 首页 | 教程导航 | 章节列表、进度指示、编号标签 |
| 教程页面 | 章节侧边栏 | 固定定位、当前章节高亮、可折叠子章节 |
| 教程页面 | 内容区域 | Markdown 渲染、代码高亮、复制按钮 |
| 速查表页面 | 搜索栏 | 全宽搜索框、实时过滤、快捷键提示 |
| 速查表页面 | 速查卡片 | 双列网格、代码片段 + 标题 + 描述、hover 边框发光 |

### 4.3 响应式设计

- 桌面优先设计，适配大屏阅读体验
- 平板端：侧边栏折叠为汉堡菜单，速查卡片单列
- 移动端：全屏内容，底部导航栏，代码块横向滚动
